# Maaltyd: AI-Assisted Recipe App

Maaltyd is an intelligent web application designed to simplify your cooking and meal preparation. It leverages AI to help you discover, create, and organize recipes. Whether you're importing recipes from your favorite food blogs or creating them from scratch, Maaltyd provides a streamlined, notecard-style experience.

## Features

*   **AI-Powered Recipe Generation:** Create simple, notecard-like recipes from web links or by typing them out from scratch.
*   **Intelligent Recipe Filtering & Sorting:** Find the perfect recipe by filtering and sorting based on ingredients you already have.
